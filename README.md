 Task 2: Data Visualization and Storytelling

 Objective
To create meaningful visualizations that tell a business story using sales data.

 Tool Used
- Power BI

 Dataset
- Superstore Dataset

 Visualizations Created
1. Sales by Category (Bar Chart)
2. Profit by Sub-Category (Column Chart)
3. Sales by Region (Map)
4. Monthly Sales Trend (Line Chart)
5. Top 10 Customers by Sales (Bar Chart)
6. Sales vs Profit (Scatter Plot)

 Key Insights
- Technology category contributes the highest sales.
- Some sub-categories generate losses and need attention.
- Sales show consistent growth over time.
- Few customers contribute major revenue.
- Regional sales performance varies significantly.


This task helped me understand how to choose the right visual, avoid clutter, and present insights in a clear business-focused manner.
